# Portafolio de Ciencia de Datos

¡Hola!

Este repositorio documenta mi viaje de aprendizaje para convertirme en una científica de datos autodidacta. Estoy siguiendo un plan de estudio estructurado en fases para construir una base sólida en programación, estadística y aplicación de modelos, con un enfoque en el mundo de los negocios.

Cada semana, este repositorio se actualizará con nuevos proyectos y ejercicios que reflejan mi progreso. Mi objetivo es demostrar mis habilidades de análisis de datos, visualización y machine learning a través de proyectos reales.

**Plan de Estudio por Fases:**

* **Fase 1: Los Fundamentos** (Semanas 1-4)
    * **Semana 1:** Fundamentos de Python y SQL.
    * **Semana 2:** Control de flujo y manipulación de datos.
    * **Semana 3:** Análisis Exploratorio de Datos (EDA) y agregación.
    * **Semana 4:** Visualización y unión de tablas.
* **Fase 2: El Corazón de la Ciencia de Datos** (Semanas 6-12)
    * Aquí me enfocaré en la creación de modelos de Machine Learning.
* **Fase 3: Conexión con el Negocio y Profesionalización** (Semanas 12-18)
    * Me centraré en el **data storytelling**, el despliegue de modelos y el uso de herramientas de visualización de datos.

¡Gracias por visitar mi portafolio!
